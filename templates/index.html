Classroom

En estos tiempos complicados, como no puedes ir al colegio, traeremos el colegio a tu casa. La meta de este proyecto es traer a las escuelas un método simple para que se pueda enviar y recibir las tareas de parte de los alumnos. Ademas de fácilmente saber como van las zonas, tanto para el maestro como para el alumno.

Tutorial

Si esta viendo esto, probablemente acaba de clonar el repositorio del código, y es la primera vez que lo esta corriendo. Esto es lo que tiene que hacer para empezar a utilizarlo en su colegio.

Detalles previos

La interfaz gráfica es bastante simple, el alumno puede ver sus tareas pendientes, terminarlas, adjuntar documentos a las mismas, y ver sus notas. El profesor puede crear tareas para su clase, y calificarlas. El administrador tendrá una cuenta de profesor, con acceso a la base de datos.

La cuenta que viene por default es con el nombre de usuario root, y con la contraseña toor. Lo primero que tiene que hacer una vez inicie sesión, es darle clic al botón de actualizar donde este la sección de maestros, y cambiar el nombre de usuario y contraseña. Ya con el sitio asegurado, empecemos a montar nuestra clase en linea.

Nota: Yo recomendaría crear 2 o 3 cuentas de administrador, para dividir el trabajo.

Creación de los grados

Al iniciar sesión, diríjase a grados -> añadir.

El grado sera nuestra unidad básica, ya que los alumnos se separaran por grados, y las clases también. Ingrese unicamente el nombre del grado. Ejemplos:

  • 1ero Primaria A
  • 3ro Básico B
  • V Bachillerato en Computación

Creación de los maestros

Al iniciar sesión, diríjase a maestros -> añadir.

Ingrese los datos que se pidan del maestro, consultando con el primero que contraseña desea. Aquí lo único que tenemos que hacer notar, es que el correo, al que ingrese al crear el maestro, se va a enviar un mensaje cada vez que un alumno complete una tarea. Si el maestro considera que esto es innecesario, ponga como correo “N/A”.
